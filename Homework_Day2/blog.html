Chào bạn! Hôm nay chúng ta sẽ cùng nhau xây dựng một ứng dụng Todo list đơn giản sử dụng HTML, CSS và JavaScript. Ứng dụng này sẽ có các chức năng cơ bản như thêm công việc, hiển thị danh sách công việc và xóa công việc đã hoàn thành. Đây là một dự án tuyệt vời để bạn bắt đầu làm quen với lập trình web front-end. Hãy cùng bắt đầu nhé!

Bước 1: Thiết lập cấu trúc HTML

Đầu tiên, chúng ta cần tạo cấu trúc HTML cho ứng dụng Todo list. Cấu trúc này sẽ bao gồm một ô nhập liệu để người dùng nhập công việc, một nút để thêm công việc và một danh sách để hiển thị các công việc đã thêm. 

<!DOCTYPE html>
<html lang="vi">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Todo List Đơn Giản</title>
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
        <div class="container">
            <h1>Todo List</h1>
            <div class="input-section">
                <input type="text" id="taskInput" placeholder="Thêm công việc mới">
                <button id="addButton">Thêm</button>
            </div>
            <ul id="taskList">
                <!-- Danh sách công việc sẽ được hiển thị ở đây -->
            </ul>
        </div>
        <script src="script.js"></script>
    </body>
</html>

Giải thích:

  • <!DOCTYPE html>: Khai báo loại tài liệu là HTML5.
  • <html lang="vi">: Thẻ gốc html với thuộc tính lang="vi" chỉ định ngôn ngữ của trang là tiếng Việt.
  • <head>: Chứa các thông tin meta của trang như bộ mã ký tự (charset="UTF-8"), cài đặt viewport cho responsive (<meta name="viewport"...>), tiêu đề trang (<title>), và liên kết đến file CSS (<link rel="stylesheet"... />).
  • <link rel="stylesheet" href="style.css">: Liên kết file CSS có tên style.css để định dạng giao diện.
  • <body>: Chứa nội dung hiển thị của trang web.
  • <div class="container">: Một container chính để bao bọc toàn bộ nội dung todo list.
  • <h1>Todo List</h1>: Tiêu đề chính của ứng dụng.
  • <div class="input-section">: Khu vực chứa ô nhập liệu và nút thêm công việc.
  • <input type="text" id="taskInput" placeholder="Thêm công việc mới">: Ô nhập liệu văn bản với id="taskInput" để JavaScript có thể truy cập, và placeholder hiển thị gợi ý.
  • <button id="addButton">Thêm</button>: Nút "Thêm" với id="addButton" để JavaScript có thể gắn sự kiện.
  • <ul id="taskList">: Danh sách không thứ tự ul với id="taskList" để hiển thị các công việc. Các công việc sẽ được thêm vào đây bằng JavaScript.
  • <!-- Danh sách công việc sẽ được hiển thị ở đây -->: Chú thích cho biết vị trí danh sách công việc sẽ hiển thị.
  • <script src="script.js"></script>: Liên kết file JavaScript có tên script.js để thêm chức năng cho ứng dụng.

Bước 2: Thêm một chút phong cách với CSS (tùy chọn)

Mặc dù không bắt buộc, việc thêm một chút CSS sẽ giúp ứng dụng của bạn trông đẹp mắt và dễ sử dụng hơn. Hãy tạo một file style.css và thêm vào đoạn code sau: 

body {
        font-family: sans-serif;
        margin: 0;
        background-color: #f4f4f4;
        display: flex;
        justify-content: center;

    h1 {
        text-align: center;
        margin-bottom: 20px;
        color: #333;
    }
    
    .input-section {
        display: flex;
        margin-bottom: 15px;
    }
    
    #taskInput {
        flex-grow: 1;
        padding: 10px;
        border: 1px solid #ddd;

    }

        padding: 10px 15px;

Giải thích:

  • body: Định dạng chung cho toàn bộ trang, bao gồm font chữ, màu nền, căn giữa nội dung theo chiều dọc và ngang.
  • .container: Định dạng cho container chính, bao gồm màu nền trắng, padding, bo góc, đổ bóng và chiều rộng cố định.
  • h1: Định dạng tiêu đề, căn giữa và màu chữ.
  • .input-section: Định dạng khu vực nhập liệu, sử dụng flexbox để sắp xếp các phần tử theo hàng ngang.
  • #taskInput: Định dạng ô nhập liệu, bao gồm padding, border, bo góc và margin bên phải.
  • #addButton: Định dạng nút "Thêm", bao gồm padding, màu nền, màu chữ, bo góc và con trỏ chuột.
  • #addButton:hover: Hiệu ứng hover cho nút "Thêm" khi di chuột vào.
  • #taskList li: Định dạng mỗi mục công việc trong danh sách, bao gồm padding, border dưới, sử dụng flexbox để căn chỉnh nội dung bên trong và khoảng cách giữa các phần tử.
  • #taskList li:last-child: Loại bỏ border dưới cho mục công việc cuối cùng.
  • .delete-button: Định dạng nút xóa, tương tự như nút "Thêm" nhưng với màu nền đỏ.
  • .delete-button:hover: Hiệu ứng hover cho nút xóa.

Bước 3: Thêm chức năng thêm công việc với JavaScript

Bây giờ là phần quan trọng nhất: thêm chức năng cho ứng dụng bằng JavaScript. Hãy tạo một file script.js và viết đoạn code sau: 

const taskInput = document.getElementById('taskInput');
const addButton = document.getElementById('addButton');
const taskList = document.getElementById('taskList');

addButton.addEventListener('click', addTask);

function addTask() {
    const taskText = taskInput.value.trim(); // Lấy giá trị từ ô input và loại bỏ khoảng trắng thừa

    if (taskText !== "") { // Kiểm tra nếu không phải là chuỗi rỗng
        const listItem = document.createElement('li'); // Tạo một phần tử li mới
        listItem.textContent = taskText; // Đặt nội dung của li là văn bản công việc

        // Thêm nút xóa cho mỗi công việc
        const deleteButton = document.createElement('button');
        deleteButton.textContent = 'Xóa';
        deleteButton.classList.add('delete-button'); // Thêm class CSS để định dạng
        deleteButton.addEventListener('click', deleteTask); // Thêm sự kiện xóa

        listItem.appendChild(deleteButton); // Thêm nút xóa vào li
        taskList.appendChild(listItem); // Thêm li vào ul (danh sách công việc)

        taskInput.value = ""; // Xóa nội dung ô input sau khi thêm công việc
    }
}

Giải thích:

  • const taskInput = document.getElementById('taskInput');: Lấy phần tử input bằng ID taskInput và gán vào biến taskInput.
  • const addButton = document.getElementById('addButton');: Lấy phần tử button bằng ID addButton và gán vào biến addButton.
  • const taskList = document.getElementById('taskList');: Lấy phần tử ul bằng ID taskList và gán vào biến taskList.
  • addButton.addEventListener('click', addTask);: Gắn một trình xử lý sự kiện click cho nút addButton. Khi nút được click, hàm addTask sẽ được gọi.
  • function addTask() { ... }: Định nghĩa hàm addTask để xử lý việc thêm công việc.
  • const taskText = taskInput.value.trim();: Lấy giá trị từ ô input (taskInput.value) và sử dụng .trim() để loại bỏ khoảng trắng thừa ở đầu và cuối chuỗi.
  • if (taskText !== "") { ... }: Kiểm tra xem taskText có phải là chuỗi rỗng không. Nếu không rỗng, thực hiện các bước tiếp theo.
  • const listItem = document.createElement('li');: Tạo một phần tử li mới bằng JavaScript.
  • listItem.textContent = taskText;: Đặt nội dung văn bản của phần tử li là giá trị của taskText (tức là công việc người dùng nhập).
  • const deleteButton = document.createElement('button');: Tạo một phần tử button mới.
  • deleteButton.textContent = 'Xóa';: Đặt nội dung văn bản của nút là "Xóa".
  • deleteButton.classList.add('delete-button');: Thêm class CSS delete-button vào nút để áp dụng các kiểu dáng đã định nghĩa trong CSS.
  • deleteButton.addEventListener('click', deleteTask);: Gắn trình xử lý sự kiện click cho nút xóa. Khi nút xóa được click, hàm deleteTask sẽ được gọi (chúng ta sẽ định nghĩa hàm này ở bước tiếp theo).
  • listItem.appendChild(deleteButton);: Thêm nút deleteButton làm con của phần tử listItem.
  • taskList.appendChild(listItem);: Thêm phần tử listItem (bao gồm cả nút xóa) làm con của phần tử taskList (danh sách công việc), do đó hiển thị công việc mới trên trang.
  • taskInput.value = "";: Đặt giá trị của ô input về chuỗi rỗng sau khi thêm công việc, giúp người dùng có thể nhập công việc mới dễ dàng.

Bước 4: Thêm chức năng xóa công việc

Cuối cùng, chúng ta cần thêm chức năng xóa công việc khi người dùng click vào nút "Xóa" bên cạnh mỗi công việc. Thêm hàm deleteTask vào file script.js: 

function deleteTask(event) {
    const deleteButton = event.target; // Lấy nút xóa đã được click
    const listItem = deleteButton.parentNode; // Lấy phần tử li cha của nút xóa
    taskList.removeChild(listItem); // Xóa phần tử li khỏi danh sách công việc
}

Giải thích:

  • function deleteTask(event) { ... }: Định nghĩa hàm deleteTask. Hàm này nhận một đối số event chứa thông tin về sự kiện đã xảy ra (ở đây là sự kiện click).
  • const deleteButton = event.target;: event.target trả về phần tử HTML đã kích hoạt sự kiện (trong trường hợp này là nút xóa). Chúng ta gán nó cho biến deleteButton.
  • const listItem = deleteButton.parentNode;: deleteButton.parentNode trả về phần tử cha của deleteButton, tức là phần tử li chứa công việc mà chúng ta muốn xóa. Chúng ta gán nó cho biến listItem.
  • taskList.removeChild(listItem);: Sử dụng taskList.removeChild(listItem) để xóa phần tử listItem khỏi danh sách công việc taskList. Điều này sẽ loại bỏ công việc đó khỏi giao diện người dùng.

Hoàn thành!

Vậy là chúng ta đã hoàn thành việc xây dựng ứng dụng Todo list đơn giản bằng HTML, CSS và JavaScript. Bây giờ bạn có thể mở file index.html bằng trình duyệt và thử nghiệm ứng dụng của mình. Bạn có thể thêm công việc, xem danh sách công việc và xóa công việc đã hoàn thành.

Đây chỉ là một ứng dụng Todo list rất cơ bản. Bạn có thể tiếp tục phát triển nó bằng cách thêm nhiều chức năng hơn như:

  • Chỉnh sửa công việc: Cho phép người dùng chỉnh sửa nội dung công việc đã thêm.
  • Đánh dấu công việc đã hoàn thành: Thêm checkbox hoặc nút để đánh dấu công việc là đã hoàn thành và có thể hiển thị công việc đã hoàn thành theo cách khác (ví dụ: gạch ngang).
  • Lưu trữ dữ liệu: Sử dụng LocalStorage hoặc các phương pháp khác để lưu trữ danh sách công việc ngay cả khi trang web được đóng và mở lại.
  • Sắp xếp công việc: Cho phép người dùng sắp xếp công việc theo thứ tự ưu tiên hoặc thời gian thêm.

Chúc bạn thành công và có những trải nghiệm thú vị với lập trình web!
